# RoadSoS Decisions

## Tech Stack
Frontend will be built using React + Vite.

## Styling
Tailwind CSS will be used for fast, modern, responsive UI.

## UI Theme
Dark Navy + Cyan + White.

Reason:
This gives a modern startup-style emergency platform look.

## Maps
Leaflet with OpenStreetMap will be used instead of Google Maps.

Reason:
Free, open-source, and suitable for hackathon MVP.

## Nearby Services
Overpass API will be used to fetch nearby services.

## AI
Gemini API will be used for first aid guidance.

## Storage
LocalStorage will be used for MVP.

## Backend
Backend is optional and will not be built in the first version unless required.

## Deployment
Frontend will be deployed on Vercel.

## Innovation Features
- AI First Aid Assistant
- Voice SOS
- Offline Emergency Mode
