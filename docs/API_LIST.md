# RoadSoS API List

## Browser Geolocation API
Purpose:
Get user's live latitude and longitude.

Used For:
- SOS Dashboard
- Nearby Services
- Share Location

## OpenStreetMap
Purpose:
Map display.

Used For:
- User location
- Nearby service markers

## Leaflet
Purpose:
Interactive map library.

Used For:
- Map rendering
- Markers
- Popups

## Overpass API
Purpose:
Fetch nearby public services from OpenStreetMap data.

Used For:
- Hospitals
- Police Stations
- Vehicle services

## Gemini API
Purpose:
AI-based first aid guidance.

Used For:
- AI First Aid Assistant

## Web Speech API
Purpose:
Voice command detection.

Used For:
- Voice SOS
